
# Theatre Audience Forecasting Model

A time series forecasting project that predicts theatre audience demand from historical data by engineering lag/rolling-date features and training supervised ML baselines (XGBoost, LightGBM, Random Forest) to forecast future attendance.

## Major Libraries Used:

- Numpy
- Pandas
- Matplotlib
- Seaborn
- scikit-learn
- XGBoost
- LightGBM


## Implementation

1. Download the dataset from Kaggle:

    **Theatre Audience Forecasting:** https://www.kaggle.com/datasets/akshmahee/theatre-audience-forecasting

2. Install the dependencies:

```bash
pip install -r Requirements.txt
```

3. Open the Jupyter notebook.

4. Run all cells sequentially.
